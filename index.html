CSS

CSS (Cascading Style Sheets) is a stylesheet language used to style web pages. It controls colors, fonts, spacing, positioning, and responsiveness.

Box Model

Every HTML element is considered a rectangular box, made up of:

  • Content: The actual content such as text or images.
  • Padding: Space between the content and the border
  • Border: A line surrounding the padding and content.
  • Margin: Space outside the border, separating the element from other elements.
